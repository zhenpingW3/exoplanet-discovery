## 1. 设计概念：两个互补视图

三种方法经过评估。两种被选中用于实现，作为互补视图：

### 概念 A：适居性漏斗（已选中 ✅）
**故事**：有多少行星可能支持生命？

```
所有 5,600+ 系外行星
         ↓
处于栖息地带的 ~280 颗（合适的温度）
         ↓
地球大小且宜居的 ~50 颗（0.8–1.5 地球半径）
         ↓
未来可探测的 ~20 颗（下一代望远镜）
```

**优点**：
- 简单、启发性的叙述（非常适合高中生）
- 清晰显示宜居行星的稀有性
- 强大的情感吸引力
- 直接回答"我们独处吗？"的问题

**在仪表板中的角色**：
- **初学者的主要视图** — 首先看到这个
- **情感钩子** — 回答"为什么这很重要？"
- **支持所有观众** — 即使不了解天体物理学，每个人都能理解这个问题

**决定**：✅ 被选中作为主要入口点，因为它立即吸引最人性化的问题

---

### 概念 B：探测偏差散点图（已拒绝）
**故事**：我们的探测方法让我们找到了什么？

```
Y轴：温度（K）
         ○ ○ ●
          ○   ●● ○        ← 热木星集群
         ○ ○●●●●●● ○     ← 易于探测
          ● ●●●●●●● ●
                 ○ ○
X轴：行星半径（地球半径）

颜色 = 探测方法  |  大小 = 行星质量  |  动画 = 年份
```

**优点**：
- 对数据科学家极好（多维度、以模式为中心）
- 可视化显示聚类和偏差
- 对天文学专业生很好（揭示系统性探测模式）

**缺点**：
- 4+ 个视觉维度对高中生来说太复杂
- 需要解释故事才能变清楚
- 没有发现加速的背景（时间维度与动画混淆）
- 不能清楚地回答"我们找到了什么类型的行星"
- 散点图密度使其难以看到类别

**决定**：❌ 被拒绝（复杂性超过收益；被时间线 + 分类取代）

---

### 概念 C：时间线 + 分类树（已选中 ✅）
**故事**：我们发现系外行星的速度有多快，找到了什么类型？

```
发现时间线（堆积面积图展示加速）
              ↓（点击"探索细节"）
行星分类（气泡图显示多样性和偏差）
```

**优点**：
- ✓ 简单入口点（时间线首先——所有观众在情感上都能接受"发现加速"）
- ✓ 渐进式复杂性（分类增加背景）
- ✓ 自然揭示偏差（时间线中的堆积颜色 + 分类中"热木星"的过度代表）
- ✓ 适用于所有三个观众（高中生获得"发现加速"洞察，天文学专业生看到方法主导，数据科学家理解采样偏差）
- ✓ 无需领域知识即可访问
- ✓ 清晰的类别标签支持钻取

**在仪表板中的角色**：
- **次要视图用于深度学习** — 通过"看如何找到它们"按钮访问
- **技术解释** — 展示*为什么*我们找到了我们找到的
- **支持天文学专业生和数据科学家** — 揭示探测偏差和方法论

**决定**：✅ 被选中作为次要视图，因为它解释漏斗结果背后的原因

## 整合：两视图仪表板设计

**用户旅程**：

```
用户到达
    ↓
[主要视图] 适居性漏斗
"我们独处吗？"
  - 漏斗可视化（5600 → 37 → 6）
  - 标题："只有~6颗行星可能支持生命"
  - 简单、情感、激发好奇心
    ↓
用户点击"为什么这么少？学习我们如何找到它们"
    ↓
[次要视图] 时间线 + 分类
"我们发现速度有多快，找到了什么类型？"
  - 发现时间线（显示加速）
  - 行星分类（显示多样性）
  - 标题："我们找到的大多数是热木星（易于探测）"
  - 技术解释，揭示偏差
    ↓
用户点击返回或"看全景"
    ↓
返回适居性漏斗（主视图）
```

**切换机制**：
- 按钮/标签："适居性漏斗" ↔ "发现深潜" ↔ "详细浏览器"
- 平滑视图转换
- 随时可访问两者

**为什么两个视图一起工作**：
1. **漏斗回答**："有多少宜居行星？"（什么）
2. **时间线+分类解释**："为什么我们找到了我们找到的？"（如何和为什么）
3. **一起讲述**：系外行星发现的完整故事——发现和背后的偏差

## 2. 目标（在最终设计提案前重申）
**主要目标**：以一种让所有三个观众（高中生、天文学专业生、数据科学家）都能同时参与的方式，展示我们发现系外行星的速度以及我们找到的类型。

**子目标**：
1. 交流发现加速的步伐（所有观众的情感钩子）
2. 揭示行星多样性（对高中生来说是反直觉的）
3. 揭示观测偏差（天文学专业生和数据科学家的关键洞察）
4. 为想深入学习的人提供学习途径，而不会对初学者造成压倒

---

## 3. 可视化选择

### 三视图交互式仪表板：适居性漏斗 + 时间线 + 分类 + 散点图

**为什么这种方法**：
- **情感入口点**：适居性漏斗立即回答"我们独处吗？"（所有观众都能连接情感）
- **渐进式发现**：用户可以深入了解以理解*为什么*宜居行星很少
- **分层解释**：从简单（漏斗）→ 变得技术性（时间线）→ 理解偏差（分类）
- **适用于所有观众**：高中生可以只看漏斗并理解答案；科学家可以探索所有细节
- **讲述完整故事**："这是我们找到的"（漏斗）→ "这是我们如何找到它的"（时间线+分类）→ "这是为什么我们找到了我们找到的"（偏差解释）

---

## 4. 故事假设
### 核心叙述：
**"我们对系外行星的发现揭示了一个多样化的宇宙，但探测偏差塑造了我们的发现。"**

### 支持问题：
1. **我们发现速度有多快？** 系外行星发现在加速，95% 在 2009 年后发现
2. **我们找到了什么类型？** 行星的多样性远远超过我们的太阳系
3. **为什么是这些类型？** 探测方法（凌星、径向速度）将我们偏向于某些行星
4. **什么是宜居的？** 只有少数行星处于"适居带"，具有地球样的条件

### 关键洞察：
我们没有发现所有系外行星的代表性样本——我们发现了最容易用当前技术探测的那些。这既造就了机会（我们可以改进探测方法）又造成盲点（我们错过了重要的行星类型）。

## 5. 详细可视化规范

### 视图 1：适居性漏斗（主要视图）

**图表类型**：动画漏斗/瀑布图  
**故事**："有多少行星可能支持生命？"

**漏斗级别**：

| 级别 | 数量 | 过滤器 | 标签 |
|------|------|--------|------|
| 所有系外行星 | 1,174 | 无 | 所有已发现的系外行星 |
| 栖息地带 | ~37 | 250–350K 温度 | 适合液态水的温度范围 |
| 地球大小且宜居 | ~6 | 0.8–1.5 地球半径 + 250–350K | 最有可能支持生命的 |
| 未来发现潜力 | ~10 | 可能被下一代望远镜探测 | 有待发现 |

**视觉设计**：
- 每个级别是一个水平条或部分，向下逐渐变窄
- 宽度 = 行星数量（直观显示稀有性）
- 颜色渐变：蓝色（宜居）→ 其他颜色代表消除的类别
- 动画：当用户滚动或点击"查看过滤器"时，条形向下滑动

**交互功能**：
- 悬停在每个级别：显示精确数量、百分比、示例行星
- 点击一个级别：显示过滤条件
- 可扩展描述："为什么我们使用 250–350K 作为栖息地带？"（教育）
- 底部按钮："想知道为什么这么少？→ 探索发现细节"

**使用的数据列**：
- `pl_eqt`（平衡温度）— 用于栖息地带过滤
- `pl_rade`（行星半径）— 用于地球大小过滤
- `pl_name`（行星名称）— 用于示例

**标题和注释**：
- 主标题："只有~6 颗行星可能支持生命这样"
- 副标题："那是所有系外行星发现中的 0.51%"
- 按钮标签："为什么这么少？点击探索我们如何找到它们 →"

**为什么这有效**：
- 高中生：立即获得情感冲击；清楚的问题答案
- 天文学专业生：看到过滤器并理解宜居性标准
- 数据科学家：理解样本缩减管道和过滤逻辑

---

### 视图 2 & 3：发现深潜（次要视图 - 时间线 + 分类 + 散点图）

#### 部分 A：发现时间线（顶部 ~40% 的仪表板）

**图表类型**：堆积面积图或堆积柱状图  
**X 轴**：发现年份（1995–2025）  
**Y 轴**：系外行星累积数量  
**堆积依据**：探测方法，具有不同颜色  
- **凌星法**：蓝色（主导，~70% 总数）
- **径向速度**：橙色（~20%）
- **成像**：绿色（~5%）
- **其他（微透镜、计时）**：灰色（~5%）

**关键特性**：
- 悬停时提示：年份、每种方法的数量、累积总数
- 参考线：2010 年（标记加速点）
- 文本注释："95% 的系外行星在 2009 年后被发现"
- 简单标题："发现在加速"

**数据列使用**：
- `disc_year`（发现年份）— 时间线 X 轴
- `discoverymethod`（凌星、径向速度等）— 时间线颜色

**为什么这有效**：
- 高中生："哇，我们最近找到了大部分！"
- 天文学专业生：可以识别每种方法何时成为主导
- 数据科学家：清楚地看到增长率和方法贡献

---

#### 部分 B：行星分类（底部 ~60% 的仪表板）

**图表类型**：气泡图或分类柱状图  
**维度**：
- **X 轴**：行星类型（基于半径 + 质量的类别）
- **Y 轴**：每种类型的行星数量
- **气泡大小**：数量（视觉强化）
- **颜色**：平均平衡温度（凉色→温色，蓝色→红色）

**行星类型类别**（基于地球比较）：
1. **超级地球**（0.8–2.0 地球半径，<10 地球质量）
2. **海王星样**（2.0–6.0 地球半径，10–100 地球质量）
3. **木星样**（6.0+ 地球半径，100+ 地球质量）
4. **热木星**（木星大小，距离恒星非常近，<1000K）
5. **地球样且宜居**（0.8–1.5 地球半径，250–350K 温度）

**数量（近似）**：
- 超级地球：~948（最常见，由凌星法发现）
- 海王星样：~131
- 木星样：~66
- 热木星：~23（由于探测偏差被过度代表）
- 地球样且宜居：~6（稀有）

**关键特性**：
- 点击一个类别：突出/过滤（钻取能力）
- 提示：类型名称、数量、百分比、示例行星
- 标签每个类别带有简单定义（"木星样 = 大质量气体巨星"）
- 突出"地球样且宜居"类别（使稀有性可见）

**使用的数据列**：
- `pl_rade`（行星半径）— 分类类别
- `pl_bmasse`（行星质量）— 分类类别
- `pl_eqt`（平衡温度）— 分类颜色、宜居性

**为什么这有效**：
- 高中生：直观看到大多数行星都很奇怪（不像地球）
- 天文学专业生：识别"热木星"过度代表是偏差指标
- 数据科学家：理解分类规则和数据分布

---

#### 部分 C：详细浏览器（散点图）

**图表类型**：交互式散点图  
**X 轴**：行星半径（地球半径，对数刻度）  
**Y 轴**：温度（K，对数刻度）或质量（地球质量，对数刻度）  
**点颜色**：探测方法（凌星 = 蓝色，径向速度 = 橙色，成像 = 绿色，其他 = 灰色）  
**地球参考**：绿色星形标记（1 地球半径，288K）  
**栖息地带**：蓝色阴影矩形（250–350K 温度范围）

**交互功能**：
- Y 轴切换：温度 vs 质量
- 颜色编码切换：探测方法 / 发现年份 / 统一视图
- 悬停显示：行星名称、恒星、半径、温度/质量、发现年份/方法
- 所有 1,174 行星都包含

**为什么这有效**：
- 高中生：看到极端行星和空白区域，理解稀有性
- 天文学专业生：看到聚类显示探测方法的优势和劣势
- 数据科学家：评估数据质量、缺失数据模式、采样偏差

## 7. 关键数据洞察

### 洞察 1：爆炸性近期增长
- 1995–2008：~350 个系外行星被发现（缓慢稳定）
- 2009–2025：~5,250 个系外行星被发现（加速）
- 2020–2025：~2,500 被发现（最快期间）
- **翻译**："在过去 5 年中，我们发现了总数的一半"

### 洞察 2：探测方法主导结果
- **凌星法**：70% 的发现（寻找行星在恒星前通过时的光线下降）
  - 最佳用于：大行星、近轨道、亮星
  - 偏差：错过小的、远距离的行星
- **径向速度**：20%（检测行星重力引起的恒星摇晃）
  - 最佳用于：大质量行星、更大轨道
  - 偏差：需要长期观测
- **其他方法**：10%（成像、微透镜、计时）
- **翻译**："您用于搜索的方法决定了您找到的内容"

### 洞察 3：大多数行星都很极端
- 热木星（理论上不应存在，但易于探测）= 23
- 超级地球（在我们太阳系中未知）= 948
- 海王星样行星 = 131
- 地球大小、宜居的行星 = ~6
- **翻译**："行星的多样性远超我们预期"

### 洞察 4：宜居行星很少
- 只有 ~5% 的系外行星温度适合液态水（250–350K）
- 只有 ~6 个既有宜居温度又是地球大小的
- **翻译**："如果生命需要地球样的条件，这就像大海捞针"

## 8. 设计原则

### 视觉层级：
1. **主要焦点**：漏斗（立即冲击，"我们独处吗？"）
2. **次要焦点**：分类（背景，"但我们找到了什么？"）
3. **支持细节**：颜色、提示、图例

### 颜色策略：
- **探测方法（时间线）**：色盲友好
  - 凌星法：蓝色 #3498db
  - 径向速度：橙色 #e74c3c
  - 成像：绿色 #2ecc71
  - 其他：灰色 #7f7f7f
- **温度（分类）**：发散刻度
  - 凉色（宜居）：蓝色 #3498db
  - 热色（极端）：红色 #e74c3c
  - 中等范围：黄色/橙色

### 交互规则：
- **漏斗**：点击展开细节；无需过滤（显示完整故事）
- **分类**：点击突出，显示示例，解释类别
- **交叉链接**：点击类型 → 在时间线中过滤以显示何时发现
- **重置按钮**：返回完整仪表板

### 文本和标签：
- 保持所有标签简单（无不解释的行话）
- 谨慎使用表情符号（💡 用于洞察，⭐ 用于亮点）
- 包括"这意味着什么？"天文学术语部分

## 9. 风险和缓解

| 风险 | 影响 | 缓解 |
|------|------|------|
| **两个图表感觉不连贯** | 用户看不到时间线和类型如何相关 | 添加交叉链接（点击类型 → 在时间线中突出）和桥接文本（"最近的发现大多是超级地球"） |
| **旧发现中缺失数据** | 早期系外行星的不完整画面 | 清楚地显示缺失数据；注意旧数据不太完整 |
| **"热木星"名称让高中生困惑** | 行话障碍 | 包括简单定义："距离其恒星非常近的木星样行星" |
| **数据一目了然时压倒** | 用户不知道从哪里开始 | 添加指导叙述：标题 → 注释 → 提示 |
| **极端行星属性难以可视化** | 16× 地球半径很抽象 | 总是与地球比较；显示"地球 = 1"作为基准 |
| **不同观众想要不同深度** | 不能用一个可视化满足所有人 | 提供带渐进式披露的提示（简单 → 详细） |
